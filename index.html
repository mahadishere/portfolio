Hotspring Whirlpools

Official Webpage
An AR application to showcase whirlpools. With Augmented Reality, before purchasing a hot tub, you can visualize whirlpools through your smartphone. This app gives real-world dimensions as if the pools were physically present.

It provides options such as seats, color, material, etc., saving significant time over traditional methods.

SolidVR

Official Release

Project goal

Optimization of the data transfer process from CAD system to VR application in real-time via direct interface.

Product Output

Typical VR solutions import CAD via exchange formats (e.g., STP/FBX), which is slow. We built a direct interface and evaluated optimization methods.

AprilTag 3 for Unity

Video Demo on Youtube

Note: Special thanks to Keijiro for the AprilTag support for Unity.

Upgrade of AprilTag 2 for Unity by Keijiro. The AprilTag 3 DLL enables multithreading (pthread) and better robustness across lighting conditions compared to the older DLL.

Key differences observed:
  • Faster and more robust detection with varying exposure.
  • Multithreading support via pthread in C++.
  • Arm64 UWP support.
Prototype targeting HoloLens 2.
